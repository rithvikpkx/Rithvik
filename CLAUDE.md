# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository. It also provides details about what this repository contains, what is being developed, and what the final goal of this project is.

## Overview

This is a personal portfolio site for Rithvik Praveen Kumar, which is going to be deployed at [rithvik.ai](https://rithvik.ai). It's goal is to serve as a "one stop shop" for all things Rithvik and to showcase his projects, skills, and personality to recruiters and any other interested parties.

## Features

Current:

- Basic static HTML and CSS website to establish visual foundation and general look and feel.

Final:

- Next.js + Supabase + Vercel deployment
- Clean, minimal, "techy" aesthetic
- Smooth, animated, high quality UX
- AI Chatbot named "RAG (powered by Rithvik Augmented Generation)" that is context aware of everything on website and can answer questions about Rithvik. Implemented with RAG pipeline.
- "I am Rithvik" button with admin login to make live production time changes and additions to website through the UI

## Rules

- Keep design philosophy simple.

- Always write a short 1-2 sentence description of functions.
- Include a short 1 line comment for any potentially confusing or convoluted sections of code.
